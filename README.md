# Autotrader-car-price-prediction
A two-stage machine learning pipeline was developed to predict used car prices using real-world Autotrader data. The project covers the full workflow, including data preprocessing, exploratory data analysis, and feature engineering to prepare the data for modelling.

In the initial stage, multiple baseline models such as Linear Regression, Decision Trees, and K-Nearest Neighbours (KNN) were implemented and compared, with performance optimisation applied to improve predictive accuracy.

Building on this foundation, the second stage focuses on advanced machine learning techniques, including ensemble methods, polynomial regression, and dimensionality reduction. Model interpretability was further enhanced using feature importance analysis and SHAP (SHapley Additive exPlanations), providing deeper insight into the key factors influencing car prices.

The dataset used in this project is sourced from Autotrader and is proprietary; therefore, it cannot be publicly shared.
